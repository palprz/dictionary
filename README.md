## Dictionary [February 2017]

Desktop application for read, write, edit and remove translations.

### Technologies and tools:
- Java 8
- Eclipse
- Maven
- JUnit 
- MongoDB
- JavaFX (with plugin e(fx)clipse and Scene Builder 8.0)
